# airbnb-data

dependencies require python 3.10  

#### Setup  
- Create virtual environment  
```
python -m venv .venv 
```  
Activate virtual environment  

- Windows:
```
source .venv/Scripts/activate
```  

- Mac:
```
source .venv/bin/activate

```  

Install dependencies   
```
pip install -r "requirements.txt"
```
